Project 2

Tic-Tac-Toe Game 🎮🏆

This Tic-Tac-Toe game is a responsive, interactive, and visually appealing web-based game where two players can compete in multiple rounds. Users can enter their names, decide the number of matches they want to play, and see a live scoreboard updating in real-time.

✨ Key Features:

Customizable Matches: Users can select how many rounds they want to play.

Live Scoreboard: Keeps track of player wins and displays the final result with an animated announcement.

Neon-Themed Design: The game board has a neon glow effect to create a visually striking experience.

Smooth Animations & Hover Effects: Interactive elements to enhance user engagement.

Fully Responsive: Works across different screen sizes, including mobile devices.

🔹 Tech Stack Used:

✅ HTML → Structure of the game

✅ CSS → Custom styling with grid layout, neon effects, and animations

✅ JavaScript → Handles game logic, user input, scoreboard updates, and animations
